<h2>🛠️ Installation Steps:</h2>

<p>1. Clone Repo</p>

```
git clone https://github.com/Asia4046/projectrc-api
```

<p>2. CD Into the repo</p>

```
cd projectrc-api
```

<p>3. Install the Deps</p>

```
npm i
```

<p>4. Configure ENV variables (.env)</p>

```
DATABASE_URL='database url' 
JWT_SECRET='jwt_secret'
```
